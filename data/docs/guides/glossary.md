# Glossary

## A

**Agent**: An AI system that can perform tasks autonomously

**API Key**: Authentication credential for API access

## C

**CAG**: Computation-Augmented Generation

**Chunk**: A segment of a document for retrieval

## E

**Embedding**: Vector representation of text

**Endpoint**: API URL for specific operations

## H

**Hybrid Agent**: Combines RAG and CAG capabilities

## L

**LLM**: Large Language Model

**Latency**: Time taken to process request

## M

**Metadata**: Additional information about documents

**MMR**: Maximal Marginal Relevance

## R

**RAG**: Retrieval-Augmented Generation

**Rate Limit**: Maximum requests per time period

## S

**Semantic Search**: Search by meaning, not keywords

**Streaming**: Real-time response delivery

## T

**Token**: Unit of text for LLM processing

**Temperature**: Randomness parameter (0-1)

## V

**Vector Store**: Database for embeddings

**Vector**: Numerical representation of data
