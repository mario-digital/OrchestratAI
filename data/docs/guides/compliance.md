# Compliance Guide

## Certifications

- SOC 2 Type II
- ISO 27001
- GDPR compliant
- HIPAA compliant (Enterprise)

## Data Residency

Choose data location:
- US (default)
- EU
- Asia-Pacific (Enterprise)

## Audit Logs

Access audit logs:
```
GET /audit-logs
```

Includes:
- API calls
- Document access
- Configuration changes
- User actions

## Data Processing Agreements

DPA available for:
- Pro plans
- Enterprise plans

Download from Settings > Legal

## Compliance Reports

Generate reports:
- Access logs
- Data processing records
- Security incidents
- Uptime reports

## Right to be Forgotten

Request data deletion:
1. Settings > Privacy
2. Request deletion
3. Confirm via email
4. Data deleted within 30 days

## Data Export

Export all data:
```
GET /export
```

Includes:
- Documents
- Execution history
- Configuration
- Metadata
