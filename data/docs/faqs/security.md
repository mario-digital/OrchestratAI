# Security FAQ

## Is my data encrypted?

Yes, all data is encrypted in transit using TLS 1.3 and at rest using AES-256.

## Where is data stored?

Data is stored in SOC 2 Type II certified data centers in the US and EU.

## Do you support SSO?

Yes, Enterprise plans include SAML 2.0 SSO support.

## How do you handle API keys?

API keys are hashed using bcrypt and never stored in plain text.

## Are you GDPR compliant?

Yes, we are fully GDPR compliant and offer data residency options.

## Do you perform security audits?

We conduct quarterly penetration testing and annual security audits.
