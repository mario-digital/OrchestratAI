# Getting Started FAQ

## How do I get started with OrchestratAI?

1. Sign up for a free account
2. Get your API key from the dashboard
3. Install the SDK: `pip install orchestratai`
4. Make your first API call

## What programming languages are supported?

We officially support Python, JavaScript/TypeScript, Go, and Java.

## Where can I find code examples?

Check out our GitHub repository for comprehensive examples and starter templates.

## How long does setup take?

Most developers are up and running within 15 minutes.

## Do I need a credit card for the free tier?

No credit card required for the free tier. Just sign up and start building.
