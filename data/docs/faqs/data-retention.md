# Data Retention FAQ

## How long do you store my data?
Account data: While active. Logs: 90 days. Billing: 7 years.

## Can I request data deletion?
Yes, submit a deletion request via Settings > Privacy.

## What happens when I delete my account?
All data is permanently deleted within 30 days.

## Do you backup my data?
Yes, encrypted backups are stored for disaster recovery.

## Can I export my data?
Yes, use Settings > Export Data or the API export endpoint.
