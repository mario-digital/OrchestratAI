# Features FAQ

## What agent types are supported?

We support RAG (Retrieval-Augmented Generation), CAG (Computation-Augmented Generation), Hybrid, and Direct agents.

## Can I customize agent behavior?

Yes, agents are fully customizable through configuration parameters and system prompts.

## Do you support streaming responses?

Yes, all agent types support SSE (Server-Sent Events) streaming.

## What LLM providers do you integrate with?

We support OpenAI, Anthropic Claude, AWS Bedrock, and custom model endpoints.

## Can I chain multiple agents together?

Absolutely. Our orchestration layer handles complex multi-agent workflows.

## Is there a limit to context window size?

Context limits depend on the underlying LLM model you're using.
